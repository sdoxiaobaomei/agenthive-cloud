## 详细成本分析

### 1. 单台 ECS (Docker Compose)

**推荐**: 阿里云新人优惠 **¥99/年** (~¥8/月)

```
阿里云 ECS (2vCPU 2GB):
├── 实例费用: ¥99/年 (新人优惠)
├── 流量费: ~¥0 (包含在套餐内)
└── 域名: ~¥60/年

总计: ~¥13/月 (含域名)
```

**资源配置**:
| 服务 | 内存限制 | 说明 |
|------|----------|------|
| Nginx | 50 MB | 反向代理 |
| Landing | 300 MB | Nuxt 3 SSR |
| API | 300 MB | Express |
| Agent Runtime | 600 MB | AI Agent |
| PostgreSQL | 300 MB | 数据库 |
| Redis | 80 MB | 缓存 |
| 系统预留 | 370 MB | OS + Buffer |

### 2. 阿里云 ACK (Kubernetes)

#### 固定成本

| 项目 | 费用 | 说明 |
|------|------|------|
| **ACK 托管版** | **¥0.82/小时** | 控制平面费用 |
| **月度固定** | **~¥600** | 无法节省 |

#### 可变成本

| 方案 | 配置 | 月费用 |
|------|------|--------|
| 最低配置 | 1×ecs.c6.large (2vCPU 4GB) | ~¥200 |
| 推荐配置 | 2×ecs.c6.large + 高可用 | ~¥400 |

**总成本**:
- 最低: **~¥800/月**
- 推荐: **~¥1000/月**

#### 演示场景优化

```bash
# 演示前10分钟启动
kubectl scale nodegroup --nodes=1

# 演示后关闭
kubectl scale nodegroup --nodes=0
```

| 场景 | 计算 | 费用 |
|------|------|------|
| 关闭状态 | ACK 控制平面 | **¥600/月** |
| 演示当天 (2小时) | ¥600 + (¥200/30/24×2) | **~¥600.6/月** |
| **仅演示4次/月** | ¥600 + ¥1 | **~¥601/月** |

### 3. AWS EKS (Kubernetes)

#### 固定成本

| 项目 | 费用 |
|------|------|
| EKS 控制平面 | **$72/月 (~¥520)** |

#### 可变成本

| 项目 | 配置 | On-Demand | Spot |
|------|------|-----------|------|
| EC2 | t3.medium | ~$30/月 | ~$9/月 |
| EBS | 50GB gp3 | ~$4/月 | ~$4/月 |

**总成本**:
- On-Demand: **~¥570/月**
- Spot: **~¥480/月**

### 4. 混合架构 (本地 + Cloudflare)

```
成本明细:
├── 域名 (.com): ¥60/年 = ¥5/月
├── Cloudflare: ¥0 (免费套餐)
├── Cloudflare Tunnel: ¥0 (无限流量)
├── 本地 K8s: ¥0 (Kind/K3s)
└── 电费: ~¥10/月 (估算)

总计: ~¥15/月
```

## 总结

| 场景 | 推荐方案 | 月成本 |
|------|---------|--------|
| **演示场景** | 阿里云 ECS (新人 ¥99/年) | **¥13** ⭐ |
| **小团队** | 阿里云 ECS 标准版 | ¥60 |
| **生产环境** | AWS EKS Spot | ¥480 |
| **极致省钱** | 混合架构 | ¥15 |

**关键洞察**:
1. K8s 控制平面费是固定开销（阿里云 ¥600/月，AWS ¥520/月）
2. 演示场景下，ECS Docker 方案最具性价比
3. 混合架构适合有稳定本地环境的用户
